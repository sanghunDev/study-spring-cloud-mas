# study-spring-cloud-mas
* 멀티묘듈로 구성된 Spring Cloud 를 이용한 MAS 프로젝트
  * MAS 템플릿을 만들기 전 스터디용 프로젝트

# 공통 개발 환경
* Build Tool : Gradle
* Java Version : 17
* Spring Boot Version : 3.1.2
